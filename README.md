# IfBlank

Return a value if the object is empty

## Installation

Add this line to your application's Gemfile:

    gem 'if_blank'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install if_blank

## Usage

### Block Usage
`obj.if_blank? { do_something }`

### Value Usage
`obj.if_blank?("some string")`
